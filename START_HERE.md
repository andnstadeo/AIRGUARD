## 🔧 If Something Goes Wrong

### Port already in use?
```bash
npm run dev -- --port 3000
```

### CSS looks broken?
```bash
npm run dev
# Wait 10 seconds
```

### Clear everything?
```bash
rm -rf node_modules
npm install --legacy-peer-deps
npm run dev
```
